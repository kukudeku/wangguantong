## 一致性约定

- 前端统一使用 `VITE_API_BASE_URL=/api`；
- Nginx `proxy_pass http://127.0.0.1:8087/;` 末尾保留 `/`，使 `/api/payment/...` 转为后端 `/payment/...`；
- 全新数据库使用 `database/wangguantong.sql`，已有数据库按增量脚本说明升级；
- 微信、支付宝默认关闭，点餐余额支付始终可用；用户端余额充值的微信/支付宝支付只有平台确认成功后才到账；
- `wangguantong-commented/` 为旧版注释学习快照，不作为当前系统部署或验收依据。

## 提交前必须填写

- 项目组长与组员姓名；
- 项目报告中的专业、班级、学号和姓名；
- 如果课程要求文件名必须使用个人学号姓名，请复制项目报告并按实际信息重命名。

## 同步说明

- 新增用户端 `POST /recharge/payment`，仅支持微信或支付宝余额充值；
- `payment_record.business_type` 区分“点餐”和“余额充值”；
- 充值只有在支付平台验签回调或主动查单确认成功后到账，同一支付单不重复入账；
- 微信、支付宝未配置真实商户资料时保持关闭，文档不将其描述为默认可直接收款。

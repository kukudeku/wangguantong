# 注释学习副本宝塔说明

`wangguantong-commented` 是历史学习快照，不是当前生产部署包。它缺少主系统的真实支付、优惠券签到、团购券、推广邀请和统一服务工单，也只有 8 张早期基础表。

请部署仓库根目录当前主系统，并使用：

- [当前宝塔部署文档](../../docs/baota-deploy.md)
- [本地与数据库升级说明](../../docs/local-and-baota.md)
- [真实支付配置](../../docs/payment-setup.md)

如仅在本机学习注释代码，请使用同目录的 [运行说明](local-and-baota.md)，并避免覆盖当前主系统数据库。
